# AIChat Node Design

## REQ-CHAT-0001: Provide a new node type: `AIChat`

### `ai_chat.gd`

Extends `Node`. Registered as a custom node type.

```gdscript
class_name AIChat extends Node

# Properties
@export_multiline var system_prompt: String = ""
@export var api_endpoint: String = ""
@export var api_key: String = ""
@export var model: String = ""

# State
var chat_history: Array[Dictionary] = []

# Signals
signal chat_started()
signal progress(chunks: Array[String])
signal chat_finished(full_response: String)
signal chat_error(error_message: String)
```

## API and Behavior

### `send_message(prompt: String, attachments: Array[String] = [])`
1. Appends a new message to `chat_history`. 
    - If `attachments` is empty, content is a simple `String` (prompt).
    - If `attachments` is NOT empty, content is an `Array` of dictionaries (multi-modal format), including the text prompt and base64-encoded image data for each supported resource path.
2. Emits `chat_started`.
3. Prepares messages for the API:
    - If `system_prompt` is not empty, it's added as the first message.
    - Appends `chat_history`.
4. Uses `AIClient` to send a streaming request.
5. Relays `AIClient.progress` signal to `AIChat.progress`.
6. Upon completion:
    - Appends the full response to `chat_history`: `{"role": "assistant", "content": full_response}`.
    - Emits `chat_finished(full_response)`.
7. Handles errors and emits `chat_error`.

### `cancel()`
Interrupts the ongoing AI request via `AIClient.cancel()`.

### `clear_history()`
Clears the `chat_history` array.

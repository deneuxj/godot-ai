## File Structure

```

в”ңв”Җв”Җ plugin.cfg                              # Plugin manifest
в”ңв”Җв”Җ ai_assistant.gd                        # Plugin entry point, registers dock & node
в”ңв”Җв”Җ agent_assisted_3d.gd                  # Custom 3D node class (AIAgentAssisted3D)
в”ңв”Җв”Җ ai_chat.gd                            # Custom chat node class (AIChat)
в”ңв”Җв”Җ agent_assisted_3d_panel.tscn          # Editor dock UI scene
в”ңв”Җв”Җ agent_assisted_3d_panel.gd            # Editor dock UI controller
в”ңв”Җв”Җ ai_client/
в”Ӯ   в”ңв”Җв”Җ ai_client.gd                      # Abstract base / interface
в”Ӯ   в””в”Җв”Җ openai_client.gd                  # OpenAI-compatible API implementation
в”ңв”Җв”Җ generator/
в”Ӯ   в”ңв”Җв”Җ prompt_builder.gd                 # Builds AI prompt from user input + textures
в”Ӯ   в”ңв”Җв”Җ script_executor.gd                # Safely loads & runs generated GDScript
в”Ӯ   в””в”Җв”Җ custom_logger.gd                  # Intercepts engine errors

в”Ӯ   в””в”Җв”Җ ai_settings.gd                    # Reads/writes project settings
в””в”Җв”Җ generated/                            # Cache directory for generated .gd/.tscn files
```

## Implementation Order

1. **Plugin scaffolding** - `plugin.cfg`, `ai_assistant.gd`, plugin registration
2. **AI settings** - `settings/ai_settings.gd`, auto-configure project settings
3. **AI client** - `ai_client/` with OpenAI-compatible streaming
4. **Prompt builder** - `generator/prompt_builder.gd` with system prompt
5. **Script executor** - `generator/script_executor.gd` with safety measures
6. **AgentAssisted3D node** - Custom node class with generation pipeline
7. **AIChat node** - Custom chat node for generic AI interaction
8. **Editor dock** - `agent_assitted_3d_panel.tscn` with UI and integration
9. **Persistence** - Cache management, prompt change detection, regeneration
10. **Testing** - Verify with LM Studio, edge cases, error handling

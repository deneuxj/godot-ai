# AgentAssisted3D Node Design

## REQ-NODE3D-0001: Provide a new 3D node type: `AgentAssisted3D`

### `agent_assisted_3d.gd`

Extends `Node3D`. Registered as a custom node type via the plugin entry point (`ai_assistant.gd`).

```gdscript
class_name AIAgentAssisted3D extends Node3D

enum GenerationMode { SCENE, NODE_SCRIPT }

# Properties (exposed in editor)
var prompt: String = ""
var texture_attachments: Array[Texture2D] = []
var generation_mode: GenerationMode = GenerationMode.SCENE
var generated_node_name: String = "GeneratedNode"
var generated_script: Script = null # Populated in NODE_SCRIPT mode

enum GenerationStatus { IDLE, GENERATING, SUCCESS, ERROR }
var generation_status: GenerationStatus = GenerationStatus.IDLE
var status_message: String = ""
var generated_code: String = "" # Last generated content (GDScript or TSCN)

# Signals
signal progress(chunks: Array[String])
signal code_updated(code: String)
```

## REQ-NODE3D-0004: Generation Flow with Error Correction Loop

### Generation Flow

The plugin no longer executes AI-generated code. It saves the AI's response as a file and either instantiates it (Scene mode) or attaches it (Node Script mode).

```gdscript
func generate():
    var max_retries = _get_project_setting("max_retries", 5)
    generation_status = GenerationStatus.GENERATING
    status_message = "Generating..."
    
    # 1. Build initial prompt with mode-specific instructions
    var messages = PromptBuilder.build(prompt, texture_attachments, generation_mode)
    
    for attempt in range(max_retries):
        # 2. Call AI (streaming)
        var content = await _call_ai(messages)
        
        # 3. Update preview
        generated_code = content
        code_updated.emit(content)
        
        if generation_status == GenerationStatus.IDLE: return # Interrupted
        
        # 4. Validate output (Parse check)
        var validation_error = _validate_output(content, generation_mode)
        
        if validation_error == null:
            # 5. Apply results
            _save_to_disk(content, generation_mode)
            _apply_result(generation_mode)
            generation_status = GenerationStatus.SUCCESS
            status_message = "Generation complete"
            return
        
        # 6. Error Correction
        status_message = "Fixing error on attempt %d/%d" % [attempt + 1, max_retries]
        messages = PromptBuilder.build_error_correction(messages, validation_error, content)

    generation_status = GenerationStatus.ERROR
    status_message = "Generation failed after %d attempts" % max_retries
```

## REQ-NODE3D-0011: Generation Modes

### Scene Mode
AI is instructed to output a valid Godot `.tscn` file.
The resulting file is saved to `res://generated/<node_name>.tscn`.
The `AgentAssisted3D` node clears its children and instantiates this scene as a child.

### Node Script Mode
AI is instructed to output a valid Godot `.gd` script extending `Node3D` (or the appropriate type).
The resulting file is saved to `res://generated/<node_name>.gd`.
The `AgentAssisted3D` node attaches this script to itself: `set_script(load(path))`.

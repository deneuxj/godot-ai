# AgentAssisted3D Node Design

## REQ-NODE3D-0001: Provide a new 3D node type: `AgentAssisted3D`

### `agent_assisted_3d.gd`

Extends `Node3D`. Registered as a custom node type via the plugin entry point (`ai_assistant.gd`).

```gdscript
class_name AgentAssisted3D extends Node3D

# Properties (exposed in editor)
var prompt: String = ""
var texture_attachments: Array[Texture2D] = []

enum GenerationStatus { IDLE, GENERATING, SUCCESS, ERROR }
var generation_status: GenerationStatus = GenerationStatus.IDLE
var status_message: String = ""
var generated_code: String = "" # Last generated GDScript

# Signals
signal progress(chunks: Array[String])
signal code_updated(code: String)
```

## REQ-NODE3D-0004: Generation Flow with Error Correction Loop

### Generation Flow

```gdscript
func generate():
    var max_retries = _get_project_setting("max_retries", 5)
    generation_status = GenerationStatus.GENERATING
    status_message = "Generating..."
    
    # ... Build initial prompt ...
    
    for attempt in range(max_retries):
        # Call AI (streaming)
        script_text = await _call_ai(messages)
        
        # Update code property immediately (REQ-EDITOR-0004)
        generated_code = script_text
        code_updated.emit(script_text)
        
        if generation_status == GenerationStatus.IDLE: return # REQ-NODE3D-0010
        
        # Execute and check for errors
        var error_result = await ScriptExecutor.execute_with_error(script_text, self)
        
        if error_result.error == null:
            success = true
            break
        
        # Append error to chat history for correction
        status_message = "Fixing error on attempt %d/%d" % [attempt + 1, max_retries]
        messages = PromptBuilder.build_error_correction(messages, error_result, script_text)
    
    # ... Finalize status ...
```

### REQ-NODE3D-0010: Interruption

The user can cancel an ongoing generation. This stops the AI client and resets the node status.

```gdscript
func cancel_generation():
    if _active_client:
        _active_client.cancel()
    generation_status = GenerationStatus.IDLE
    status_message = "Generation cancelled"
    generation_finished.emit()
```
